HTML5

¿Qué es HTML5?:
HTML5 es la quinta revisión mportante del lenguaje básico de la World Wide Web, HTML. HTML5 específica dos variantes de sintaxis para HTML: una «clásica», HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML 5 que deberá servirse con sintaxis XML (application/xhtml+xml).

Nuevos elementos de HTML5:

HTML5 establece una serie de nuevos elementos y atributos que reflejan el uso típico de los sitios web modernos. Algunos de ellos son técnicamente similares a las etiquetas DIV y SPAN, pero tienen un significado semántico, como por ejemplo NAV (bloque de navegación del sitio web) y FOOTER.

  • HEADER: Se utiliza para contener la cabecera de un sitio
  • FOOTER: Contiene el pie de página de un sitio
  • NAV: Contiene la funcionalidad de navegación para la página
  • ARTICLE:Contiene una pieza intependiente de contenido, que tendría sentido sindicar como elemento RSS. Ejemplo: Una noticia.
  • SECTION: Se utiliza bien para agrupar artículos en diferentes temas, o bien para definir las diferentes secciones de un solo artículo
  • TIME: Se usa para marcar tiempos y fechas
  • ASIDE: Define un bloque de contenido relacionado con el contenido principal que lo rodea, pero que no es esencial para el flujo del mismo
  • HGROUP: Se utiliza para incluir más de un encabezado si quieres que cuente como un único encabezado en la estructura de encabezado de la página
  • FIGURE y FIGCAPTION: Se usan para encapsular una figura como un único elemento y contener el título de la figura, respectivamente.

Mi experiencia con diseño web

El diseño web es parte fundamental de las tecnologías del siglo XXI, y el uso de HTML5 es necesario dentro del diseño.
Mi experiencia con HTML5 va desde algún tiempo, y este trabajo me sirvió como forma de recordar lo básico referente al maquetado Web, sin embargo, estas formas nativas y primitivas se vuelven tediosas si ya se conoce framworks y SDKs que permiten automatizar algunas de las acciones que se deben hacer de forma manual dentro de HTML nativo.
Puedo decir que este ejercicio sirvió como forma de refrescamiento de conocimientos.
